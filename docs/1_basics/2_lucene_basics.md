# Apache Lucene 기초

## 1. Lucene 소개
- 오픈소스 기반의 검색 라이브러리
- 검색 엔진의 기본 기능 제공
  - 색인
  - 검색
  - 형태소 분석

## 2. 주요 특징
- 고성능 정보 검색(IR) 라이브러리
- 소프트웨어에 검색 기능 쉽게 추가 가능
- Elasticsearch, Solr 등의 기반 기술

## 3. 핵심 기능
### 색인 기능
- 문서 파싱
- Analyzer를 통한 분석
- 검색용 인덱스 생성

### 검색 기능
- 전문 검색 (Full Text Search) 지원
- 긴 문장 내 부분 검색 가능
- 효율적인 검색 알고리즘
